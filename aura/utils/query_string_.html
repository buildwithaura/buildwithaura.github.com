query_string

Converts a hash into a query string.

Usage

query_string(hash)

Example

query_string(:q => "peanuts", :page => 2)
#=> q=peanuts&page=2

query_string(:fruits => %w[apple banana])
#=> q[]=apple&q[]=banana
