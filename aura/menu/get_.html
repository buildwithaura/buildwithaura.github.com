    get

    Returns a menu item of a given key.

    Usage

    Aura.admin.menu.get(keyname)

    Example

    Get and set

    Use Aura::Menu#add to add, then get to get.

    Aura.admin.menu.add "hello",
  :name => "Hello",
  :href => '/admin/lol',
  :icon => 'settings'

item = Aura.admin.menu.get('hello')  #=> #<MenuItem>
item.name                             #=> "Hello"
item.icon                             #=> "settings"
