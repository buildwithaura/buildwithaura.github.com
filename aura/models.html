Models

The namespace for models.

Creating a model

Subclass Model (Aura::Models) in the Aura::Models namespace.

module Aura::Models
  class Movie < Model
    # ...
  end
end

Finding models

Use all (Aura::Models).

Aura::Models.all.map  |m| m.name

Find by name

Use get (Aura::Models) or Ruby's const_get.

Aura::Models.get('contact_form')       #=> Aura::Models::ContactForm
Aura::Models.const_get(:ContactForm)   #=> Aura::Models::ContactForm
